Deep Learning Channel estimation in 5G and future networks that operate in mmWave and mMIMO systems

This research work looked at the importance of m-MIMO in 5G wireless networks for fast communication. The more antennas used, the faster the data can be transmitted, but it becomes expensive to figure out how to use each antenna correctly. Deep learning can help determine the best way to use each antenna. This work compares the Deep Learning approach to traditional methods such as LS and LMMSE estimation and it was found out that the deep learning approach performs better and can handle a large number of antennas without increasing complexity. DL approach can be implemented on parallel architecture hardware, such as GPU or FPGA, to improve processing time. The test results demonstrated DL's superiority over traditional methods in terms of BER and received power signal.

Development of a machine learning plant growth estimator for hydroponics

In this research project I developed a plant growth estimator for broadleaf plants grown under hydroponics. I compared three approaches: the use of traditional machine learning (combining morphological operators, feature descriptors and classification algorithm) to come up with a computer vision model, the use of deep learning models (CNNs) and a third approach of using a pre-trained deep learning model as a feature extractor and then concatenate it with a machine learning algorithm to come up with an estimation model. State of the art results with high generalisation were achieved using the third approach.
